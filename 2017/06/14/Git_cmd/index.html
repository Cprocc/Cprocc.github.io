Git命令
Git版本控制软件

  • git clone 将远程仓库的代码克隆到本地
  • git status 输出工作目录的状态，看到所有编辑过的文件
  • git add 在目录下新建一个文件
  • git add -A 所有更改过的文件都会被添加到暂存区
  • git add afilename.txt 添加某个文件到暂存区
  • git commit -m “your message” 对于暂存区的文件进行描述
  • git push 上传到远程仓库
  • git checkout – afilename.txt 还原更改
  • gie reset –hard 所有东西重置为origin状态
