hexo写博文时插入本地图片

摘要
使用hexo时为博文加入图片

在github+hexo搭建好的博客下写博文时，发现.md文档编写时没有办法直接加入本地图片，加入文件的在本地的相对url，再部署到github上时，图片加载不出来。因为本地hexo博客部署到github上时文件的结构已经发生了变化

我认为可能可行的解决办法

1.将图片文件夹传到云服务器上，或者在github.io文件下建立一个文件夹

未进行实际操作

2.本地引用，利用CodeFalling/hexo-asset-image插件

  1. 将本地blog文件夹下中的_config.yml中的post_asset_folder属性改为true
  2. 在hexo下输入npm install https://github.com/CodeFalling/hexo-asset-image –save，安装插件
  3. 安装完毕之后用hexo 新建博文的时候在本地_posts页面中生成.md文件的同时会生成一个同名的文件夹用来放置图片。经测试这个文件夹也可以手动生成
  4. 在编辑.md文档是按照要求填写图片url路径即可
    例如
    ![](blog_hexo+github\Git_success.jpg)
    注意
  • 路径中不要出现中文，会出现问题，也就意味着文件名和图片名都不要出现中文
